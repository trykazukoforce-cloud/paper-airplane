# かみヒコーキ ✈️

紙飛行機をできるだけ遠くまで飛ばす、ブラウザだけで遊べるミニゲームです。HTML / CSS / JavaScript の1ファイル（[index.html](index.html)）だけで動きます。

## 遊び方

1. [index.html](index.html) をブラウザで開きます（ダブルクリックでOK）。
2. 画面を**下方向にドラッグ**して、発射の**角度**と**パワー**を決めます。
3. 指（マウス）を**はなすと発射**。点線のガイドが飛ぶ軌道の目安です。
4. 着地すると飛んだ距離が記録されます。クリックでもう一度挑戦できます。

## 特徴

- **上昇気流リング**: 黄色いリングを通ると上向きの揚力とスピードがもらえ、より遠くへ飛べます。
- **風**: 発射ごとに「追い風 / 向かい風 / 無風」が変化します。画面右下に表示されます。
- **ベスト記録**: これまでの最高距離はブラウザに保存され、次回も引き継がれます。
- **マウス・タッチ両対応**: PCでもスマホでも遊べます。

## 動かし方

インストールやビルドは不要です。

- かんたんに試す: [index.html](index.html) をブラウザにドラッグ&ドロップ、またはダブルクリック。
- ローカルサーバーで開く場合（任意）:

  ```bash
  # Python が入っていれば
  python -m http.server 8000
  # ブラウザで http://localhost:8000 を開く
  ```

## 技術

- HTML5 Canvas による描画
- `requestAnimationFrame` を使ったゲームループ
- 重力・空気抵抗・風を取り入れたシンプルな物理シミュレーション
- 外部ライブラリ・依存なし
